# Путь художника в геймдев

Практический мини курс из 6 уроков. Видео + текст. Для начинающих художников которые хотят в игровую индустрию.

## Для кого этот курс
- для начинающих художников, которые хотят разобраться, как устроен путь в геймдев
- для тех, кто хочет двигаться в сторону 2D art, concept art или смежных ролей
- для тех, кому нужен понятный старт без хаотичного поиска информации по разным роликам и постам

## Формат
- 6 уроков
- видео-уроки и текст
- материалы можно проходить в своем темпе
- доступ останется навсегда
